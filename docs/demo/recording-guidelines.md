# Recording Guidelines — Campus Study Hub Demo

## Video Specifications

- **Duration**: 2–3 minutes
- **Resolution**: 1920×1080 (1080p)
- **Format**: MP4 (H.264)
- **Frame rate**: 30 fps

## Recommended Tools

- **Screen recording**: OBS Studio (free), Loom, or QuickTime (macOS)
- **Audio**: Use a headset mic or external microphone; avoid laptop speakers
- **Browser**: Chrome or Firefox with a clean profile (no bookmarks bar, no extensions visible)

## Pre-recording Checklist

- [ ] Backend running with seed data loaded
- [ ] Browser at 100% zoom, clean tab bar
- [ ] Notifications silenced (Do Not Disturb mode)
- [ ] Close other applications
- [ ] Test audio levels before recording

## Frame Order

1. **Login page** → enter credentials → click Login
2. **Dashboard** → pause 2 seconds to show layout
3. **Semester view** → click into Semester 1
4. **Subject detail** → scroll through notes, papers, videos
5. **Download a note** → show download dialog
6. **Book a room** → fill form → submit
7. **Admin approval** → approve the booking
8. **Search** → type query → show results
9. **Health check** → show `/actuator/health` response
10. **Wrap-up** → return to dashboard

## Narration Tips

- Speak clearly and at a moderate pace
- Pause briefly between sections for editing
- Mention feature names explicitly ("Now I'll show the room booking feature")
- Keep mouse movements smooth and deliberate

## Post-recording

- Trim dead air at start and end
- Add title card (optional): "Campus Study Hub — Demo"
- Export as MP4 and upload to project docs or YouTube (unlisted)
